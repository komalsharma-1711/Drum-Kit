# Drum-Kit
A responsive Drum Kit application built with HTML, CSS, and JavaScript. Play drum sounds using mouse clicks or keyboard keys, with real-time button animations and audio effects. This project showcases JavaScript event listeners, DOM manipulation, and interactive UI development.
# Features
• Play drum sounds using mouse clicks.
• Play drum sounds using keyboard keys (W, A, S, D, J, K, L).
• Interactive button animations for visual feedback.
• Audio playback using JavaScript.
• Responsive and user-friendly interface.
